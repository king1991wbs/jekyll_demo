我们在写程序的时候，有时候会遇到需要操作一组bool值的情况。例如我们有这样一组考试成绩{77，90，80，59，85，30}， 要记录这6个同学在某此考试中是否通过的情况，最简单可以用这样的一个{1,1,1,0,1,0}列表来表达。 但是，在具体编程中，我们要怎么去存储这样的一组数据呢？ 我估计很数人的第一反映是这样做：

// bool pass[6] = {1,1,1,0,1,0}; //

当然，这也是我的第一反应。。。这的确是最直接的方法，不是么？

但是，当我今天读《optiming cpp》这本书的boolean vector operation这一节时，我才知道还有更加巧妙的办法。书本的原文摘录在这里，供读者参考：

An integer may be used as a Boolean vector.For example,if a and b are 32-bit integers,then the expression y = a & b; will make 32 AND-operations in just one clock cycle.The operators &,|,^,~ are useful for Boolean vector operations.

这段话的意思就是说我们可以用一个integer类型的整数去表达一个整数数组或者说向量。

读完这段话后，突然感觉心情有点小激动。。。

用这种方法，我们上面的代码就可以改写成这样：

// int pass = 58; //

这个做法真是不能在巧了不是吗？

当然，有的同学会说“那我要使用数组中的某个值时该怎么办？”

这个当然没问题啦。我们在之后的程序中如果需要用数组的某个之的话，就可以同过&操作来得到我们想要的值。比如，现在我需要知道第二个同学 {对应数组列表中从右数第二个元素}是否通过，我们就可以这样：

// int second = pass & 2； //

这样子我们就可以知道第二个同学是否通过了不是吗？我们还可以通过>>以为操作符来一次获得数组中的每一个元素值。。。 好了，先写道这里吧，日后有时间再完善以下。。。
